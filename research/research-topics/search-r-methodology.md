# SEARCH-R方法论体系研究

## 📋 研究背景

SEARCH-R是一个研究方法论框架，探索如何利用LLM和Research Agent辅助人类开展研究探索。

**背景描述**：
- LLM具有强大的知识和推理能力
- Research Agent可以系统化地执行研究任务
- 需要方法论来指导如何有效结合两者辅助人类研究

**研究动机**：
- 建立系统化的研究方法论
- 设计高效的Research Agent
- 提供可复用的研究框架

---

## 🎯 研究目标

建立完整的SEARCH-R方法论体系，包括研究循环、Agent设计、文档标准等。

### 核心问题

1. **SEARCH-R研究循环的本质是什么？**
   - 为什么重要：循环是方法论的核心
   - 期望答案：循环的理论基础和设计原则

2. **如何设计Research Agent？**
   - 为什么重要：Agent是研究执行者
   - 期望答案：Agent架构、核心能力、记忆系统

3. **如何保证研究质量？**
   - 为什么重要：研究需要质量保证
   - 期望答案：质量门控机制、元认知意识

4. **如何文档化研究过程？**
   - 为什么重要：可追溯性和可复用性
   - 期望答案：文档标准、模板体系

### 期望产出

- [ ] **SEARCH-R方法论体系**：完整的研究方法论框架
- [ ] **Research Agent框架**：Agent设计和实现指南
- [ ] **文档模板体系**：标准化的文档模板
- [ ] **Skills库**：可复用的研究能力模块

### 成功标准

- 方法论体系完整且可用
- Agent框架可指导具体实现
- 模板体系经过实践验证
