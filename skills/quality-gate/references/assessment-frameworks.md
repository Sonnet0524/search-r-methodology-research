# 评估框架指南

## 什么是研究质量评估？

研究质量评估是判断研究成果价值和可信度的过程。

## 评估维度

### 确定性评估

| 等级 | 标准 | 表现 |
|------|------|------|
| HIGH | 逻辑清晰，证据充分 | 结论可靠 |
| MEDIUM | 逻辑基本清晰，证据部分缺失 | 需要补充验证 |
| LOW | 逻辑不清晰或证据严重缺失 | 需要重新研究 |

### 可接受性评估

| 等级 | 标准 | 表现 |
|------|------|------|
| HIGH | 符合预期，可直接应用 | 继续推进 |
| MEDIUM | 基本符合，需要小幅调整 | 优化后应用 |
| LOW | 不符合预期，需要重大调整 | 需要Human决策 |

### 认知混淆判断

- **无混淆**: 理解一致，无认知偏差
- **有混淆**: 存在理解歧义或概念不清

## 评估流程

```
研究成果
    ↓
确定性评估
    ↓
可接受性评估
    ↓
混淆判断
    ↓
综合评估
    ↓
决策建议
```

## 评估工具

### 质量门控矩阵

```
         可接受性HIGH  可接受性MEDIUM  可接受性LOW
确定性HIGH    继续         小优化        大调整
确定性MEDIUM  验证         验证          重做
确定性LOW     重做         重做          重做
```

### 决策规则

```
IF 确定性=HIGH AND 可接受性=HIGH AND 无混淆:
    → NONE: 无需介入

IF 存在混淆:
    → CLARIFICATION: 请求澄清

IF 确定性=MEDIUM OR 可接受性=MEDIUM:
    → DIRECTION: 请求方向指导

IF 确定性=LOW OR 可接受性=LOW:
    → DECISION: 请求Human决策
```
