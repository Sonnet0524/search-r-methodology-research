# 验证方法指南

## 什么是理论验证？

理论验证是通过证据和逻辑检验理论有效性的过程。

## 验证类型

### 内部一致性验证

检验理论内部是否自洽。

**检查点**：
- 概念定义是否清晰
- 命题之间是否矛盾
- 推导逻辑是否正确

### 外部一致性验证

检验理论是否符合已知事实。

**检查点**：
- 是否与现有知识冲突
- 是否能解释已知现象
- 是否有反例存在

### 实践验证

通过实践检验理论有效性。

**检查点**：
- 是否可操作
- 是否产生预期效果
- 是否有边界情况

## 验证流程

```
理论框架
    ↓
设计验证方案
    ↓
收集证据
    ↓
分析证据
    ↓
得出结论
    ↓
修正理论（如需要）
```

## 验证方法

| 方法 | 适用场景 | 说明 |
|------|---------|------|
| 逻辑推演 | 理论内部 | 检查推理正确性 |
| 案例验证 | 实践层面 | 用具体案例检验 |
| 对比分析 | 理论层面 | 与现有理论对比 |
| 实验验证 | 可操作时 | 设计实验检验 |
