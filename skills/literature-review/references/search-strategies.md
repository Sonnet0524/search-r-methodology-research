# 文献检索策略

## 检索渠道优先级

| 渠道 | 适用场景 | 优先级 |
|------|---------|:------:|
| 百度学术 | 中英文学术文献 | ⭐⭐⭐⭐⭐ |
| 百度搜索 | 技术文档、博客 | ⭐⭐⭐⭐ |
| 百度百科 | 概念定义 | ⭐⭐⭐ |

## 检索流程

```
1. 关键词提取
   ↓
2. 学术文献检索（百度学术）
   ↓
3. 网络资源补充（百度搜索）
   ↓
4. 概念澄清（百度百科）
   ↓
5. 整理归纳
```

## 关键词扩展技巧

- 同义词扩展
- 中英文对照
- 领域术语

## 质量筛选标准

- 来源权威性
- 时效性（近3年优先）
- 引用数量
- 发表平台
