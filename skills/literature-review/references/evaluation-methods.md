# 文献评估方法

## 评估维度

| 维度 | 高质量标准 | 低质量表现 |
|------|-----------|-----------|
| 来源 | 顶级会议/期刊、知名机构 | 不明来源、个人博客 |
| 方法 | 研究方法清晰、可复现 | 方法模糊、不可复现 |
| 数据 | 数据充分、来源可靠 | 数据不足、来源不明 |
| 结论 | 结论有据、逻辑清晰 | 结论武断、缺乏依据 |

## 评估流程

```
1. 来源评估 → 是否权威？
2. 方法评估 → 是否科学？
3. 数据评估 → 是否充分？
4. 结论评估 → 是否有据？
```

## 快速判断技巧

- 查看作者背景
- 检查引用来源
- 评估研究方法
- 验证结论逻辑
