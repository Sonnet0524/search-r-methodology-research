# 假设形成指南

## 什么是研究假设？

研究假设是对研究问题的尝试性回答，是可以被验证的命题。

## 假设形成流程

```
观察现象
    ↓
识别问题
    ↓
提出假设
    ↓
设计验证
    ↓
收集证据
    ↓
验证/修正
```

## 好假设的特征

| 特征 | 说明 | 示例 |
|------|------|------|
| 可验证性 | 能够被证据支持或反驳 | ✅ 使用工具A比工具B效率高 |
| 具体性 | 有明确的变量和关系 | ✅ X与Y成正相关 |
| 可操作性 | 可以设计验证方法 | ✅ 通过实验对比验证 |

## 假设类型

### 描述性假设
描述现象或状态
- 例：当前系统存在X问题

### 解释性假设
解释现象原因
- 例：X问题的原因是Y

### 预测性假设
预测未来趋势
- 例：采用Z方案可以解决X问题

## 假设表达模板

```markdown
## 假设: [假设名称]

### 假设陈述
[清晰的假设陈述]

### 依据来源
- 观察1: ...
- 文献2: ...

### 可验证指标
- 指标1: ...
- 指标2: ...

### 验证方法
- 方法: ...

### 预期结果
- 如果假设成立: ...
- 如果假设不成立: ...
```

## 注意事项

1. **避免过度概括** - 不要从单一案例得出普遍结论
2. **考虑替代解释** - 同一现象可能有多种解释
3. **保持开放心态** - 证据可能支持或反驳假设
4. **迭代修正** - 根据验证结果调整假设
